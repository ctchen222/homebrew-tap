# Homebrew Tap

Custom Homebrew formulae by ctchen222.

## Installation

```bash
brew tap ctchen222/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| chronoflow | Terminal-based calendar and todo manager |

## Install a Formula

```bash
brew install ctchen222/tap/chronoflow
```

Or after tapping:

```bash
brew install chronoflow
```
